Read the chapters to understand how they work. Run the code to see them in action.

## Running

Install [Rust](https://rustup.rs), then:

```
./start.sh
```

Open [http://127.0.0.1:8080](http://127.0.0.1:8080). Press `Ctrl-C` to stop everything.
